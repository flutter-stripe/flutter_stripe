# stripe_identity_android

[![pub package](https://img.shields.io/pub/v/stripe_identity_android.svg)](https://pub.dev/packages/stripe_identity_android)

The Android implementation of the [`flutter_stripe_identity`](https://github.com/flutter-stripe/flutter_stripe/tree/main/packages/stripe_identity) plugin.

## Usage

This package is the endorsed Android implementation of `flutter_stripe_identity`. It gets automatically added to your dependencies by adding the `flutter_stripe_identity` package to your `pubspec.yaml`:

```yaml
dependencies:
  flutter_stripe_identity:
```
