# stripe_identity_platform_interface

[![pub package](https://img.shields.io/pub/v/stripe_identity_platform_interface.svg)](https://pub.dev/packages/stripe_identity_platform_interface)

A common platform interface for the [`flutter_stripe_identity`](https://github.com/flutter-stripe/flutter_stripe/tree/main/packages/stripe_identity) plugin.

This interface allows platform-specific implementations of the `flutter_stripe_identity` plugin, as well as the plugin itself, to ensure they are supporting the same interface.
